# React.js Application

A simple flight booking application.

## Features

- View Flights
- View booked Flight
- Filter Flight
- Add seat booking with 2 min reservation
- Login
- Registration

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/prashanta0234/flight-client
   cd flight-client
   ```
2. Install the project
   ```bash
   npm i
   ```
3. Run the project
   ```bash
   npm start
   ```
